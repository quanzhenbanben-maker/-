
🍺 飲み会幹事コンシェルジュ
飲み会の幹事が最適なお店を素早く見つけるための、社内向け飲食店検索・管理ツールです。

機能
・お店を探す — キーワード検索＋絞り込み条件（エリア・予算・個室・ジャンルなど）でお店を検索
・店舗を登録する — ホットペッパーのURLを貼るだけで店舗情報を自動取得・登録
・レビューを投稿する — 実際に行ったお店の感想・金額・目的などを社内で共有


注意事項
・.env ファイルは絶対にGitHubにアップしないでください
・APIキーは開発メンバーに直接共有してください
・本アプリは社内利用を想定しています
